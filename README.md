## Estructura Basica - Proposito 2025 
- 01_Primeros_pasos_Python

1.  **01_CursoPython:**
    * **Descripción:** Este curso  abarca los fundamentos de la programación en Python, cubriendo temas esenciales para iniciar en el lenguaje.

2.  **02_Comprehensions_Funciones_Manejo_Errores:**
    * **Descripción:** Un curso enfocado en técnicas avanzadas de Python como las "comprehensions" (list, dict, set), el uso y creación de funciones, y la gestión de errores mediante bloques `try-except`.

3.  **03_Curso_Python_PIP_Entornos_Virtuales:**
    * **Descripción:** Este módulo se centra en la administración de paquetes con PIP (el gestor de paquetes de Python) y la creación y gestión de entornos virtuales, esenciales para el desarrollo de proyectos aislados en Python.

4.  **04_CursoUnittestingPython:**
    * **Descripción:** Un curso dedicado a las pruebas unitarias en Python, utilizando el módulo `unittest` u otras librerías para asegurar la calidad y el correcto funcionamiento del código.

5.  **05_CursoPatronesDisenioSOLIDPython:**
    * **Descripción:** Este curso profundiza en los patrones de diseño de software y los principios SOLID (Single Responsibility, Open/Closed, Liskov Substitution, Interface Segregation, Dependency Inversion) aplicados en el contexto de Python, mejorando la estructura y mantenibilidad del código.

6.  **06_Curso_Django:**
    * **Descripción:** Un curso introductorio o intermedio sobre Django, el popular framework web de Python, cubriendo la creación de aplicaciones web, modelos, vistas, plantillas y URLs.

7.  **07_Curso_Django_Rest_Framework:**
    * **Descripción:** Este curso se especializa en la construcción de APIs RESTful utilizando Django Rest Framework, una potente extensión de Django para crear servicios web robustos y escalables.

8.  **08_Curso_FastAPI:**
    * **Descripción:** Un curso sobre FastAPI, un moderno y rápido framework web para construir APIs con Python, destacando su rendimiento, tipado y documentación automática con OpenAPI.

9.  **09_Curso_Flask:**
    * **Descripción:** Este curso cubre Flask, un microframework web ligero de Python, ideal para el desarrollo rápido de aplicaciones web y APIs más pequeñas o con requisitos específicos.

10. **10_CursoComplejidadAlgoritmicaPythondd:**
    * **Descripción:** Un curso dedicado al análisis de la complejidad algorítmica en Python, estudiando cómo medir la eficiencia de los algoritmos (notación Big O) y optimizar el rendimiento del código.

11. **12_CursoDespliegueAplicacionesPythonNube:**
    * **Descripción:** Este curso se enfoca en el despliegue de aplicaciones Python en entornos de nube,  utilizando servicios como AWS cubriendo temas como contenedores (Docker) y servicios serverless.

12. **13_CursoEstructurasDatosLinealesPython:**
    * **Descripción:** Un curso fundamental sobre estructuras de datos lineales en Python, como listas, pilas, colas y listas enlazadas, y su implementación y uso eficiente en la resolución de problemas.
